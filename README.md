# Repositorio del Trabajo de Fin de Grado

## Descripción
Este repositorio contiene el código y los archivos de configuración utilizados
en el desarrollo del prototipo del Trabajo de Fin de Grado.

## Estructura
El contenido se organiza por componentes del sistema, incluyendo dispositivos
de campo, gateway y servicios de comunicación.
